# Two Pointers

## Overview

The two-pointers pattern uses two pointers to iterate through data structures, typically to find pairs or optimize search operations in sorted/ordered data.

## Pattern Characteristics

- **When to use**: Linear data structures (arrays, lists etc.), input with predictable dynamic (sorted array, palindromic string), finding pairs, reversing sequences
- **Advantages**: O(1) space, efficient O(n) time
- **Common variations**: Inward traversal (opposite-direction), unidirectional traversal (same-direction), staged traversal (sliding window)
